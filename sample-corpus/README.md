# Sample Corpus

This folder documents the neutral sample layer used by the public Stonewall
showcase. It is intentionally small: the public repository demonstrates the
platform shape, not the underlying litigation corpus.

Recommended sample labels:

- `Matter 01`, `Matter 02`, `Matter 03`
- `Lead Operator`, `Coverage Desk`, `Counsel Desk`
- `NEGOTIATION-LANE`, `MEDICAL-REVIEW`, `CHRONOLOGY-LOCK`

Use these labels in demos, fixtures, screenshots, and generated JSON snapshots.
Private source material belongs in the private repository, not this showcase.
