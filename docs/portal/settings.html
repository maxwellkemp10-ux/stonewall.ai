Danger zone

Irreversible operations. These actions affect your local case cache and index files.

Clear local case cache Deletes the cached cases.json snapshot. Requires a fresh refresh-cases run before the next ingest.
Reset pipeline preferences Restores all pipeline settings to their factory defaults.
Purge local index Removes the local file index used by verify_repo_consistency.py. A full consistency run will rebuild it.
